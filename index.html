Создание "Умного Цветка" в Blender 4.5

Полное руководство по моделированию и скульптингу

1.1. Создание сцены и настройка

1 Запустите Blender 4.5
2 Выделите стандартный куб и удалите его: X → Delete
3 Сохраните проект: File → Save As → имя "Smart_Flower"

1.2. Стебель через Sculpt Mode (Scaffold)

Добавление меша Scaffold

1 Нажмите Shift + A → Mesh → Scaffold

Переход в режим скульптинга

2 Верхнее меню → Sculpting или Ctrl + Tab → Sculpt Mode

Настройка кистей

3 На панели слева выберите Draw Sharp
4 F - размер кисти, Shift + F - сила кисти
5 Проведите по сетке, чтобы вытянуть изогнутый стебель

1.3. Бутон через Geometry Nodes

1 Вернитесь в Object Mode (Tab)
2 Добавьте сферу: Shift + A → Mesh → UV Sphere

Добавление модификатора Geometry Nodes

3 Правая панель → Modifier Properties → Add Modifier → GeometryNodes

Инструменты и горячие клавиши

  • Draw Sharp - вытягивание формы
  • Clay Strips - создание рельефа
  • Elastic Deform - плавное изгибание
  • Mask (M) - создание масок
  • Inflate - раздувание поверхности
  • Ctrl + I - инвертировать маску
  • Shift + A - добавить объект
  • Tab - переключение режимов
  • Ctrl + Tab - выбор режима

Советы

  • Часто сохраняйтесь (Ctrl + S)
  • Используйте числовые клавиши для быстрого доступа к слоям
  • Экспериментируйте с силой и размером кистей
  • Не бойтесь отменять действия (Ctrl + Z)
